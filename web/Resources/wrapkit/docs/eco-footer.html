Wrapkit Footer

Usage

var myFooter = new WrapkitFooter( '#myFooter', options );

Options

Name Type Default Descriptions
rtlMode boolean false set the Footer direction to rtl/ltr.
skin string default

set the Footer skin. Available skins:

default, dark, light, red, orange, yellow, green, teal, cyan, blue, violet.

setter

var opts = {
  rtlMode: false,
  skin: "default"
};
myFooter.option( opts );

getter

var options = myFooter.options;
get individually
var skin = myFooter.options.skin;

Methods

rtl()

document.querySelector("#myButton")
  .addEventListener( "click", function(){
  // set the footer direction
  myFooter.rtl(true);
});

setSkin()

document.querySelector("#myButton")
  .addEventListener( "click", function(){
  // set the footer skin
  myFooter.setSkin('light');
});

rtl

setSkin
