Wrapkit Utils

Wrapkit Utils is a collections of javascript methods that using by each objects on ecosystem. You can using it too.

Methods list

Name Usage Description
noop WrapkitUtils.noop() An empty function, use this empty function when you wish to pass around a function that will do nothing.
extend WrapkitUtils.extend(obj1, obj2) Merge the contents of two objects together into the first object.
makeArray WrapkitUtils.makeArray(obj) Turn element or nodeList into an array
jqEmiter WrapkitUtils.jqEmiter(el, type, data) Trigger event to jQuery if any.
isMobile WrapkitUtils.isMobile() Detect a Device. Return true if device is mobile and false if desktop.
viewPort WrapkitUtils.viewPort() Detect the real viewport. return width and height.
offsets WrapkitUtils.offsets(el) Retrieve the position (x,y) of an element
debounce WrapkitUtils.debounce(fn, wait, immediate) Returns a function, that, as long as it continues to be invoked, will not be triggered. The function will be called after it stops being called for N milliseconds. If `immediate` is passed, trigger the function on the leading edge, instead of the trailing.
getWidth WrapkitUtils.getWidth(el) Retrieve the the width of an element
getHeight WrapkitUtils.getHeight(el) Retrieve the the height of an element
getDocWidth WrapkitUtils.getDocWidth() Retrieve the width of document.
getDocHeight WrapkitUtils.getDocHeight() Retrieve the height of document.
createScript WrapkitUtils.createScript() Create/inject a script to document body, usefull for http request.
initSlimScroll WrapkitUtils.initSlimScroll($el, height) Init jquery slimScroll on any $elem
destroySlimScroll WrapkitUtils.destroySlimScroll($el) Destroy jquery slimScroll on any $elem
closest WrapkitUtils.closest(el, selector) Matches a CSS selector by traversing up the DOM tree.
transitionEnd WrapkitUtils.transitionEnd() Listen the end of CSS transition of an element.
