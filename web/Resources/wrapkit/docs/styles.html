Intro

This is an easy way to get started with wrapkit. All css files that you get the is the result of compiling of SCSS file. We have divided it into small pieces which are easy to customized. It's a perfect way to customize the template.

What you get?

All CSS files is a minified, but we include SCSS files that compose them.

bootstrap.css

The default and custom Bootstrap CSS. They are the result compile of bootstrap.scss and bootstrap-custom.scss. You will find that they are really easy to read.

dependencies.css

The stylesheet of required dependencies (including icons). The SCSS files:

  • font-awesome.scss
  • simple-line-icons.scss
  • payment-webfont.scss
  • weather-icons.scss
  • elegant-icon.scss
  • switchery.scss
  • loaders.scss
  • toastr.scss

components.css

Contains the styles of plugins that we use on the template:

  • typeahead.scss
  • tagsinput.scss
  • select2.scss
  • selectboxit.scss
  • daterangepicker.scss
  • minicolors.scss
  • dropzone.scss
  • fileinput.scss
  • xeditable.scss
  • wizard.scss
  • summernote.scss
  • jcrop.scss
  • morris.scss
  • easy-pie-chart.scss
  • jstree.scss
  • clndr.scss
  • jqvmap.scss

wrapkit.css

The core styles of template:

  • layout.scss
  • header.scss
  • sidebar.scss
  • content.scss
  • footer.scss
  • setup-modal.scss
  • utils.scss

pages.css

Just a custom styles that used on wrapkit pages like profile, timeline, etc:

  • signin.scss
  • locked.scss
  • error.scss
  • help.scss
  • pricing.scss
  • gallery.scss
  • photoswipe.scss
  • photoswipe-skin.scss
  • timeline.scss
  • inbox.scss
